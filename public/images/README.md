# Space Hub Images

This folder contains all the space-related images used throughout the Space Hub website.

## Image Sources (Recommended)
You can download free, high-quality space images from these sources:

### NASA Image Library (Public Domain)
- https://images.nasa.gov/
- All NASA images are public domain

### ESA/Hubble (Creative Commons)
- https://esahubble.org/images/
- Check individual image licenses

### Unsplash (Free to use)
- https://unsplash.com/s/photos/space
- Free for commercial and personal use

- Full images: 800x600px or larger (JPG)

## Quick Download Script
You can use these NASA API endpoints to get real space images:
- NASA APOD API: https://api.nasa.gov/planetary/apod
- Mars Rover Photos: https://api.nasa.gov/mars-photos/api/v1/rovers/curiosity/photos
